# Book Lab API

Book Lab API is a back-end side of Book Lab app.

## Packages Installation

Use the package manager NPM (v18.15.0 is desirable).

```bash
npm install
```

## Start the app (dev)

```bash
npm run dev
```

## Build the app

```bash
npm run build
```

## Run linter

```bash
npm run lint
```

## Format code

```bash
npm run format
```
